# Mora v0.09 trait 泛型设计

> **状态**：设计文档（v0.08.5 收官后）。v0.09 = trait 泛型 + impl 泛型 + where 子句 + 单态化。
> **设计原则**：渐进式扩展——保持 v0.08 trait 系统完全向后兼容；新能力只增不改。

## 0. 背景

v0.08 trait 系统（commit `01c3015`）支持：

- trait 定义（`trait Name [: Parent, ...]`）
- 默认实现（`fn name(params) = expr` / `do ... end`）
- 继承（`trait Foo: Bar, Baz`）
- self-having / self-less 方法
- dyn dispatch（`Trait::new("ForType")`）

**v0.08 局限**（按规划书第 4 章 5-8 周）：

1. trait 不接受泛型参数
2. impl 不接受泛型参数
3. 没有 where 子句约束
4. 只能 dyn dispatch，不能单态化（性能 + 类型擦除）

## 1. 目标

v0.09 推 **3 项能力 + 1 项优化**：

| 任务 | 名称 | 优先级 | 工作量 |
|------|------|--------|--------|
| 3 | trait 泛型 `trait Foo<T>` | 🔴 高 | 1-2 周 |
| 4 | impl 泛型 `impl Foo<T> for Bar` | 🔴 高 | 1 周 |
| 5 | where 子句 | 🟡 中 | 1 周 |
| 6 | 单态化（可选） | 🟢 低 | 2-3 周 |

v0.09 = 任务 3+4+5（trait/impl 泛型 + where）；任务 6（单态化）拆为 v0.10。

## 2. 核心概念

### 2.1 泛型 trait 定义

```mora
-- trait 带一个类型参数 T
trait Container<T>
  fn get(self) -> T
  fn set(self, value: T)
end

-- trait 带多个泛型 + bound（where 简化）
trait Map<K, V>
  fn put(self, key: K, value: V)
  fn get(self, key: K) -> V
end
```

### 2.2 泛型 impl

```mora
-- impl 为具体类型 T 提供泛型 trait
impl Container<number> for BoxedNum
  fn get(self) = self.value
  fn set(self, value) = self.value = value
end

-- impl 为另一个泛型类型提供泛型 trait
impl Container<T> for Boxed<T>
  fn get(self) = self.value
  fn set(self, value) = self.value = value
end
```

### 2.3 where 子句

```mora
-- where 约束类型参数
fn sort<T>(list: list<T>) -> list<T> where T: Comparable
  -- ...
end

-- trait impl 也支持 where
impl Container<T> for Wrapper<T> where T: Clone
  fn get(self) = self.value.clone()
  fn set(self, value) = self.value = value
end
```

### 2.4 调用方式

```mora
-- 显式类型参数
let c: dyn Container<number> = Container::new("BoxedNum")
let v = c.get()  -- v 类型为 number

-- 类型推断（v0.09 不做 / v0.10 做）
let c = Container<number>::new("BoxedNum")  -- 显式
```

## 4. Type 系统扩展

### 4.1 `Type::Trait` 携带泛型

```rust
pub enum Type {
    // v0.08: Trait(String)
    // v0.09 改为：
    Trait {
        name: String,
        generics: Vec<Type>,
    },
    // 删 Type::Struct（v0.08.5 已删）
    // ...其他不变
}
```

### 4.2 `Type::Struct` 替代品：`Type::Concrete`

```rust
pub enum Type {
    // 替代 v0.08.5 删的 Struct：具体类型（带 trait 列表 + 泛型参数）
    Concrete {
        name: String,
        generics: Vec<Type>,
        traits: Vec<Type>,  // 实现的 trait（含泛型实例化）
    },
    // ...
}
```

### 4.3 Trait 泛型实例化键

`impl_method_key` / `default_impl_method_key` 扩展支持泛型：

```rust
pub(crate) fn impl_method_key(
    trait_name: &str,
    trait_generics: &[Type],
    for_type: &str,
    for_generics: &[Type],
    method: &str,
) -> String {
    // 格式: __impl_<Trait>_<TraitGen>_<ForType>_<ForGen>_<method>
    // TraitGen: "T=number,U=string"（键值对）
    // 简化: 只保留 Type 名字（`String` / `Number`）
    let tg = trait_generics.iter().map(type_name).collect::<Vec<_>>().join(",");
    let fg = for_generics.iter().map(type_name).collect::<Vec<_>>().join(",");
    format!("__impl_{}_{}_{}_{}_{}", trait_name, tg, for_type, fg, method)
}
```

**示例**：
- `Container<number>` for `BoxedNum` → `__impl_Container_number_BoxedNum__get`
- `Map<string, number>` for `DictMap` → `__impl_Map_string,number_DictMap__get`

### 5.2 impl 泛型


```rust
fn parse_impl_def(&mut self) -> Stmt {
    let generics = if self.match_token(&[TokenType::LBracket]) {
        self.parse_generic_params()
    } else { vec![] };
    let trait_name = self.consume_identifier(...);
    let trait_generics = if self.match_token(&[TokenType::LBracket]) {
        self.parse_type_list()
    } else { vec![] };
    self.consume(For, ...);
    let for_type = self.consume_identifier(...);
    let for_generics = if self.match_token(&[TokenType::LBracket]) {
        self.parse_type_list()
    } else { vec![] };
    let where_clause = if self.match_token(&[TokenType::Where]) {
        self.parse_where_clause()
    } else { vec![] };
    // ...
}
```

## 8. 完整示例

### 8.1 Container trait（带泛型）

```mora
-- container.mora
trait Container<T>
  fn get(self) -> T
  fn set(self, value: T)
end

impl Container<number> for BoxedNum
  fn get(self) = self.value
  fn set(self, value) = self.value = value
end

task main()
  let c: dyn Container<number> = Container<number>::new("BoxedNum")
  c.set(42)
  print(c.get())  -- 输出 42
end
```

### 8.2 泛型 + where

```mora
-- generic_with_where.mora
trait Comparable
  fn less(self, other: Self) -> bool
end

trait Container<T> where T: Comparable
  fn min(self, other: T) -> T
end

impl Container<number> for NumPair where number: Comparable
  fn min(self, other) = if self.less(other) then self else other
end

impl Comparable for number
  fn less(self, other) = self < other
end

task main()
  let p: dyn Container<number> = Container<number>::new("NumPair")
  print(p.min(5))  -- 5
end
```

### 8.3 嵌套泛型

```mora
-- nested.mora
trait Boxed<T>
  fn unbox(self) -> T
end

trait Container<T>
  fn get(self) -> T
end

-- Boxed<Boxed<number>> —— 双层泛型
impl Container<Boxed<number>> for Outer
  fn get(self) = self.inner
end

task main()
  let c: dyn Container<Boxed<number>> = Container<Boxed<number>>::new("Outer")
  let inner = c.get()
  print(inner.unbox())  -- 输出 box 内的 number
end
```

## 9. v0.09 不做的（留给 v0.10+）

- ❌ **单态化**（泛型代码直接编译成具体类型副本）—— 仍用 dyn dispatch
- ❌ **类型推断**（`let x = container.get()` 自动推断 x 类型）—— 需显式标注
- ❌ **高阶 trait bound (HRTB)** —— v0.09 不支持
- ❌ **关联类型**（`type Item`）—— v0.10+
- ❌ **trait object safety 自动检查**（v0.09 手动限制哪些 trait 可作 dyn）
- ❌ **泛型 task**（`task foo<T>(x: T)`）—— 与 trait 泛型独立，留 v0.11
