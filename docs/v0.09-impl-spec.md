## 2. AST 改动（src/ast.rs）

### 2.1 新增 `GenericParam` 结构

```rust
/// v0.09: 泛型参数（trait/impl/method 的类型参数）
///   例如 `trait Foo<T>` / `impl<T> Foo<T> for Bar` 中的 `T`
///   v0.09 简化: bound 是 trait 名（如 "Comparable"），v0.10 改为 Vec<TraitRef>
#[derive(Debug, Clone, PartialEq)]
pub struct GenericParam {
    pub name: String,
    /// v0.09: bound 作为字符串（trait 名），v0.10 强化为结构化
    pub bound: Option<String>,
    pub span: Span,
}
```

### 2.2 扩展 `Stmt::TraitDef`

```rust
TraitDef {
```

### 3.1 `Type::Trait` 改 struct variant

```rust
pub enum Type {
```

### 3.2 `Type::from_hint` 解析 `dyn:Foo<T>`

```rust
pub fn from_hint(hint: &str) -> Type {
    if let Some(rest) = hint.strip_prefix("dyn:") {
        // 解析 `Foo` 或 `Foo<T,U>`
        if let Some(lt) = rest.find('<') {
            let name = rest[..lt].to_string();
            let generics_str = &rest[lt+1..rest.len()-1];  // strip < >
            let generics: Vec<Type> = generics_str
                .split(',')
                .map(|s| Type::from_hint(s.trim()))
                .collect();
            Type::Trait { name, generics }
        } else {
            Type::Trait { name: rest.to_string(), generics: vec![] }
        }
    } else {
        // ... 已有逻辑
    }
}
```

### 3.3 `Type::name()` 含泛型

```rust
pub fn name(&self) -> String {
    match self {
        Type::Trait { name, generics } => {
            if generics.is_empty() {
                format!("dyn:{}", name)
            } else {
                let gs: Vec<String> = generics.iter().map(|g| g.name()).collect();
                format!("dyn:{}<{}>", name, gs.join(","))
            }
        }
        Type::Concrete { name, generics, .. } => {
            if generics.is_empty() {
                name.clone()
            } else {
                let gs: Vec<String> = generics.iter().map(|g| g.name()).collect();
                format!("{}<{}>", name, gs.join(","))
            }
        }
        // ... 已有
    }
}
```

### 3.4 `Type::compatible_with` 含泛型比较

```rust
pub fn compatible_with(&self, expected: &Type) -> bool {
    if matches!(self, Type::Any) || matches!(expected, Type::Any) { return true; }
    if matches!(self, Type::Nil) { return true; }
    
    // v0.09: Trait 泛型比较
    if let (Type::Trait { name: a, generics: ga }, Type::Trait { name: b, generics: gb }) = (self, expected) {
        if a != b { return false; }
        if ga.len() != gb.len() { return false; }
        for (x, y) in ga.iter().zip(gb.iter()) {
            if !x.compatible_with(y) { return false; }
        }
        return true;
    }
    
    // v0.09: Concrete 实现 Trait 检查
    if let (Type::Concrete { name: c, generics: gc, traits: ct }, Type::Trait { name: tn, generics: tg }) = (self, expected) {
        for t in ct {
            if let Type::Trait { name, generics } = t {
                if name == tn && generics.len() == tg.len() {
                    // 简化检查: 泛型实例化要匹配
                    return true;
                }
            }
        }
        return false;
    }
    
    // 已有逻辑
    self == expected
}
```

## 4. Parser 改动（src/parser.rs）

### 4.1 三个新函数

#### `parse_generic_params()`

```rust
/// 解析 `<T>` 或 `<T: Bound>` 或 `<T, U, V: Bound>`
/// 返回: `Vec<GenericParam>`
fn parse_generic_params(&mut self) -> Result<Vec<GenericParam>, String> {
    self.consume(&TokenType::Less, "Expected '<' for generic params")?;
    let mut params = vec![self.parse_single_generic_param()?];
    while self.match_token(&[TokenType::Comma]) {
        params.push(self.parse_single_generic_param()?);
    }
    self.consume(&TokenType::Greater, "Expected '>' after generic params")?;
    Ok(params)
}

fn parse_single_generic_param(&mut self) -> Result<GenericParam, String> {
    let span = self.span_of_previous_keyword();
    let name = self.consume_identifier("Expected generic param name")?;
    let bound = if self.match_token(&[TokenType::Colon]) {
        Some(self.consume_identifier("Expected bound trait name after ':'")?)
    } else { None };
    Ok(GenericParam { name, bound, span })
}
```

#### `parse_type_list()`

```rust
/// 解析 `<T, U, V>` 用于 trait_generics / for_generics
/// 返回: `Vec<String>` (类型名，作为 string 表示)
fn parse_type_list(&mut self) -> Result<Vec<String>, String> {
    self.consume(&TokenType::Less, "Expected '<' for type list")?;
    let mut types = vec![self.consume_identifier("Expected type name")?];
    while self.match_token(&[TokenType::Comma]) {
        types.push(self.consume_identifier("Expected type name")?);
    }
    self.consume(&TokenType::Greater, "Expected '>' after type list")?;
    Ok(types)
}
```

#### `parse_where_clause()`

```rust
/// 解析 `where T: Bound, U: Bound2`
fn parse_where_clause(&mut self) -> Result<Vec<GenericParam>, String> {
    self.consume(&TokenType::Where, "Expected 'where' keyword")?;
    let mut clauses = vec![self.parse_single_generic_param()?];
    while self.match_token(&[TokenType::Comma]) {
        clauses.push(self.parse_single_generic_param()?);
    }
    Ok(clauses)
}
```

### 4.2 `parse_trait_def` 加 generics

```rust
fn parse_trait_def(&mut self) -> Stmt {
    let span = self.span_of_previous_keyword();
    let name = self.consume_identifier("Expected trait name");
    
    // v0.09: 解析 trait 自身的泛型参数
    let generics = if self.check(&TokenType::Less) {
        self.parse_generic_params().unwrap_or_else(|e| { self.error(e); vec![] })
    } else { vec![] };
    
    // 已有: 解析 parents
    let parents = if self.match_token(&[TokenType::Colon]) { /* ... */ } else { vec![] };
    
    // ... methods 解析不变 ...
    
    Stmt::TraitDef { name, generics, parents, methods, span }
}
```

### 4.3 `parse_impl_def` 加全泛型 + where

```rust
fn parse_impl_def(&mut self) -> Stmt {
    let span = self.span_of_previous_keyword();
    
    // v0.09: impl 自身的泛型参数 `impl<T, U>`
    let generics = if self.check(&TokenType::Less) {
        self.parse_generic_params().unwrap_or_else(|e| { self.error(e); vec![] })
    } else { vec![] };
    
    let trait_name = self.consume_identifier("Expected trait name after 'impl'");
    
    // v0.09: trait 的泛型参数 `Foo<T>`
    let trait_generics = if self.check(&TokenType::Less) {
        self.parse_type_list().unwrap_or_else(|e| { self.error(e); vec![] })
    } else { vec![] };
    
    self.consume(&TokenType::For, "Expected 'for' after trait name in impl");
    let for_type = self.consume_identifier("Expected type name after 'for'");
    
    // v0.09: for_type 的泛型参数 `Bar<U>`
    let for_generics = if self.check(&TokenType::Less) {
        self.parse_type_list().unwrap_or_else(|e| { self.error(e); vec![] })
    } else { vec![] };
    
    // v0.09: where 子句
    let where_clause = if self.match_token(&[TokenType::Where]) {
        self.parse_where_clause().unwrap_or_else(|e| { self.error(e); vec![] })
    } else { vec![] };
    
    // ... methods 解析不变 ...
    
    Stmt::ImplDef {
        generics, trait_generics, trait_name, for_type,
        for_generics, where_clause, methods, span,
    }
}
```

### 4.5 lexer 复用

`<T>` 的 `<` 和 `>` 与比较运算符同一 token（`Less` / `Greater`）。

**消歧规则**：
- `<` 后面紧跟 `IDENT` → 泛型开始
- `<` 后面是表达式 → 比较运算符

parser 通过 lookahead (`self.peek()`) 决定。

## 5. 向后兼容策略

### 5.1 AST literal 迁移

所有现有测试代码（`src/interpreter.rs` test 模块、`examples/*.mora` 解析结果构造）需补字段。

**TraitDef 迁移**：
```rust
// v0.08
Stmt::TraitDef { name, parents, methods, span }

// v0.09
Stmt::TraitDef { name, generics: vec![], parents, methods, span }
```

**ImplDef 迁移**：
```rust
Stmt::ImplDef {
    generics: vec![],
    trait_generics: vec![],
    trait_name,
    for_type,
    for_generics: vec![],
    where_clause: vec![],
    methods,
    span,
}
```

**DynTrait 迁移**：
```rust
Expr::DynTrait { generics: vec![], trait_name, span }
```

### 5.2 推荐：使用 `..Default::default()` 语法

```rust
Stmt::TraitDef {
    name,
    parents,
    methods,
    span,
    ..Default::default()  // generics 默认 vec![]
}
```

需要为 `Stmt` enum 加 `#[derive(Default)]`（如果还没有）。所有 variant 的字段都需 `Default`（`Vec<T>` 默认空 vec、`Option<T>` 默认 None、`bool` 默认 false、`String` 默认空串）。
